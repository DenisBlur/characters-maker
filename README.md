# characters-maker

## Install & Run Dev
1. Clone repository
   
HTTPS https://github.com/DenisBlur/characters-maker.git
```
git clone https://github.com/DenisBlur/characters-maker.git
```

2. Move to characters-maker dir

```
cd characters-maker
```

3. Install dependencies & Run project

```
$ npm i
$ npm run dev
```
